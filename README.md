# UniApp-Cat-Rebuild 项目说明

## 项目简介
### 项目用途
这是给杭外猫守社写的猫咪名册小程序。新手第一次写前端，请多多包容，也欢迎各位大佬指正。
### 技术栈
技术栈采用uniapp+wot-design-uni，本来想加上vue-router的但是奈何适配不好，用v-if手写了页面切换逻辑。
### 主要功能
- list.vue 展示信息
- detail.vue 点进去的详细信息
- support.vue 支持页面
- feedback.vue 反馈页面


## 开始使用

### 环境要求
- Node.js >= 16.x
- HBuilderX 或 VS Code 开发工具
- uni-app 框架依赖

然后用hbuilderx打开就行了

**PS:本项目目前使用的是模拟数据，由ai生成(懒)。所以看起来有些唐。要用的话可以修改data.js里面的内容以及index.vue里面的调用**
